# JavaScript & TypeScript Learning Repository

A polished educational repository for exploring JavaScript, TypeScript, CSS, and algorithm/data structure concepts with practical examples and learning notes.

## Repository Structure

- `cascading-style-sheets/`
  - Practical CSS notes covering selectors, pseudo-classes, pseudo-elements, and common CSS functions.

- `javascript-sheet/`
  - JavaScript reference files organized by topic, including fundamentals, modern syntax, DOM/event handling, utilities, and algorithm support code.

- `js-for-algorithms/`
  - Algorithm and data structure content with explanatory notes.
  - Includes implementations for sorting, searching, recursion, dynamic programming, and interview-style problems.
  - Contains selected LeetCode solutions and data structure examples.

- `typescript-sheet/`
  - A progressive TypeScript learning path from basic concepts to advanced patterns.
  - Covers type systems, interfaces, generics, modules, decorators, error handling, testing, React integration, and Node/Express examples.
  - Includes supporting configuration files such as `tsconfig.json` and `jest.config.js`.

## Objective

This repository is designed to be a reliable study and reference resource for developers who want to:

- Learn core JavaScript and TypeScript concepts with real examples.
- Practice algorithms and data structures in a structured way.
- Build confidence transitioning from JavaScript to advanced TypeScript development.
- Maintain a portable knowledge base for web development fundamentals.

## How to Use This Repository

1. Review the folder structure to locate the topic you want to learn.
2. Open the relevant `.js`, `.ts`, or `.md` file to read code examples and explanations.
3. For TypeScript files, compile and run using the TypeScript compiler or test using `jest` when applicable.

## Notes

- This repository is intended for learning and personal development.
- Content is organized by topic and progression rather than production architecture.
- The collection focuses on strengthening programming fundamentals and applied problem solving.
