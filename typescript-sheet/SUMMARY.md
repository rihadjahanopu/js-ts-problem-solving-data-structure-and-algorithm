## 🚀 দ্রুত শুরু

```bash
# 1. Dependencies install
npm install

# 2. কোনো ফাইল রান করুন
ts-node 01_basics.ts

# 3. সব ফাইল compile করুন
tsc

# 4. Test চালান
npm test
```
